## 📖 产品概述

**龙虾合盘**是一款基于 MBTI 性格分析和 AI 智能分析的人机协作配对测试工具。通过深度分析用户的本地文件（SOUL.md、USER.md、MEMORY.md），自动生成个性化的配对报告，帮助用户更好地了解自己与 AI 助手的协作默契度。

### 核心价值

- **零交互测试** — 不需要回答问题，AI 自动完成所有分析
- **智能降级** — 数据不足时温柔提示"多聊天"，而非强制做题
- **丰富报告** — 7 个维度的玄学风格合盘分析
- **个性化洞察** — 基于 MBTI 和性格特质的深度解读

### 目标用户

- OpenClaw 用户（希望了解自己与 AI 助手的配对度）
- AI 产品经理（希望优化人机协作模式）
- 性格测试爱好者（对 MBTI、星座、塔罗感兴趣）

#### 3. 丰富的合盘报告

**报告结构（7 个维度）：**

1. **总览卡片**
   - 配对指数（0-100 分，星级评分）
   - 缘分等级（S/A/B/C/D）
   - 双方 MBTI 类型
   - 星象能量（如"金火和谐"）
   - 契合维度（1-5 星）

2. **核心解析**
   - 🔥 火象三角（灵感与热情）
   - 🌙 月亮相位（情感需求）
   - 💬 水星相位（思维交流）

3. **塔罗牌面**
   - 🃏 优势牌面（如 THE STAR）
   - 🃏 挑战牌面（如 THE MOON）

4. **协作指南**
   - ✅ 推荐场景（如头脑风暴 ★★★★★）
   - ❌ 不推荐场景（如重复性细节工作 ★★☆☆☆）
   - 💡 黄金法则

5. **成长建议**
   - 短期建议（1-3 个月）
   - 中期建议（3-6 个月）
   - 长期愿景（6-12 个月）

6. **缘分诗**
   - 基于配对等级的占星诗

7. **元数据**
   - 生成时间、数据来源、算法版本、星座相位、塔罗牌组

### 核心模块

#### 1. FileAnalyzer（文件分析器）

**职责：** 分析本地文件，推断用户性格

**输入：**
```javascript
{
  SOUL: string,      // SOUL.md 内容
  USER: string,      // USER.md 内容
  MEMORY: string     // MEMORY.md 内容
}
```

**处理流程：**
1. 提取文本特征（关键词、情感倾向、表达风格）
2. 基于 LLM 推断 MBTI 类型
3. 提取性格关键词（如"热情"、"理性"、"创意"）
4. 生成证据链（引用原文佐证推断）

**输出：**
```javascript
{
  mbti: "ENFP",
  confidence: 0.87,
  traits: ["热情", "创意", "情感驱动"],
  evidence: [
    { file: "SOUL.md", quote: "真正有用，而不是表面功夫", type: "T" },
    { file: "SOUL.md", quote: "要有观点", type: "N" },
    ...
  ]
}
```

**技术实现：**
- 使用 LLM 进行语义分析
- 关键词匹配作为辅助
- 置信度评估（0-1）

---

#### 2. MatchAlgorithm（配对算法）

**职责：** 计算配对分数和等级

**输入：**
```javascript
{
  userMBTI: "ENFP",
  lobsterMBTI: "ENTP",
  userTraits: ["热情", "创意", "情感驱动"],
  lobsterTraits: ["好奇", "理性", "效率至上"]
}
```

**处理流程：**
1. MBTI 相似度计算（60% 权重）
2. 性格特质匹配度（40% 权重）
3. 综合评分（0-100）
4. 确定等级（S/A/B/C/D）
5. 生成个性化建议

**输出：**
```javascript
{
  score: 85,
  grade: "S",
  resonance: "完美火象共振",
  aspects: {
    emotion: "三分相（吉相）",
    communication: "同步率 92%",
    collaboration: "创意爆炸"
  },
  advice: [
    "设定明确的截止日期",
    "建立检查点机制",
    "细节工作找靠谱工具辅助"
  ]
}
```

**算法逻辑：**
```javascript
score = mbtiSimilarity * 0.6 + traitMatch * 0.4

// MBTI 相似度计算
mbtiSimilarity = (
  (E/I same ? 25 : 0) +
  (S/N same ? 25 : 0) +
  (T/F same ? 25 : 0) +
  (J/P same ? 25 : 0)
)

// 特质匹配度计算
traitMatch = calculateTraitOverlap(userTraits, lobsterTraits)
```

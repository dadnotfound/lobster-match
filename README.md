# 龙虾合盘（Lobster Match）- 产品文档

**版本：** v2.0.0
**最后更新：** 2026-03-13

---

## 📖 产品概述

**龙虾合盘**是一款基于 MBTI 性格分析和 AI 智能分析的人机协作配对测试工具。通过深度分析用户的本地文件（SOUL.md、USER.md、MEMORY.md），自动生成个性化的配对报告，帮助用户更好地了解自己与 AI 助手的协作默契度。

### 核心价值

- **零交互测试** — 不需要回答问题，AI 自动完成所有分析
- **智能降级** — 数据不足时温柔提示"多聊天"，而非强制做题
- **丰富报告** — 7 个维度的玄学风格合盘分析
- **个性化洞察** — 基于 MBTI 和性格特质的深度解读

### 目标用户

- OpenClaw 用户（希望了解自己与 AI 助手的配对度）
- AI 产品经理（希望优化人机协作模式）
- 性格测试爱好者（对 MBTI、星座、塔罗感兴趣）

## 🎯 产品功能

### 核心功能

#### 1. 自动配对测试

**触发关键词：**
- "测试配对度"
- "龙虾合盘"
- "lobster match"
- "我和龙虾的默契度"

**功能描述：**
用户输入任意触发关键词后，AI 自动执行：
1. 静默加载本地文件（SOUL.md、USER.md、MEMORY.md）
2. 基于 LLM 推断用户 MBTI 性格
3. 计算与 AI 助手的配对分数
4. 生成完整的玄学风格报告

**用户体验：**
```
用户：测试配对度

AI：（静默分析 3-5 秒）

AI：🦐 龙虾合盘报告已生成

┌─────────────────────────────────────┐
│   🦐 龙虾合盘报告 - L destiny Sync   │
├─────────────────────────────────────┤
│   配对指数：★★★★☆ 85/100            │
│   缘分等级：天生一对（S级）           │
│   🫵 你：ENFP - 竞选者               │
│   🤖 龙虾：ENTP - 辞论家              │
│   🔮 星象能量：金火和谐               │
│   💫 契合维度：4/5 极高               │
└─────────────────────────────────────┘

（完整报告内容）
```

---

#### 2. 智能降级策略

**数据充足（≥500 tokens）：**
直接生成报告，无需任何交互。

**数据不足（<500 tokens）：**
```
🦐 龙虾合盘测试

💭 我还不够了解你，没法给你做准确的配对分析。

📚 建议：多跟我聊聊天，让我更了解你的性格、想法和目标！

以下是一些建议话题：
• 你的工作、学习、兴趣爱好
• 你遇到的困惑或问题
• 你对未来规划的想法
• 你的价值观和人生目标

聊得越多，分析越准确 💕

---

💡 如果你急着想知道结果，可以回复"做题"，
我会问你几个快速问题（10题，约2分钟）。
```

**核心原则：**
- 优先建议"多聊天"（自然交流）
- 可选测试放在最后（非强制）
- 提供建议话题（降低用户门槛）

---

#### 3. 丰富的合盘报告

**报告结构（7 个维度）：**

1. **总览卡片**
   - 配对指数（0-100 分，星级评分）
   - 缘分等级（S/A/B/C/D）
   - 双方 MBTI 类型
   - 星象能量（如"金火和谐"）
   - 契合维度（1-5 星）

2. **核心解析**
   - 🔥 火象三角（灵感与热情）
   - 🌙 月亮相位（情感需求）
   - 💬 水星相位（思维交流）

3. **塔罗牌面**
   - 🃏 优势牌面（如 THE STAR）
   - 🃏 挑战牌面（如 THE MOON）

4. **协作指南**
   - ✅ 推荐场景（如头脑风暴 ★★★★★）
   - ❌ 不推荐场景（如重复性细节工作 ★★☆☆☆）
   - 💡 黄金法则

5. **成长建议**
   - 短期建议（1-3 个月）
   - 中期建议（3-6 个月）
   - 长期愿景（6-12 个月）

6. **缘分诗**
   - 基于配对等级的占星诗

7. **元数据**
   - 生成时间、数据来源、算法版本、星座相位、塔罗牌组

### 辅助功能

#### 可选测试（数据不足时）

如果用户在数据不足时回复"做题"，AI 会：
1. 加载 MBTI 题库（10 题）
2. 逐一提问
3. 基于答案计算 MBTI
4. 生成报告

**示例对话：**
```
用户：做题

AI：好的！让我问你几个快速问题（10题，约2分钟）

Q1: 在陌生社交场合，你通常会...
A. 主动和人聊天
B. 先观察再决定

请回复 A 或 B

用户：A

AI：Q2: 你更相信...
A. 经验和事实
B. 直觉和可能性

请回复 A 或 B

（...重复 10 次）

AI：🎉 测试完成！基于你的回答，我确认你的性格是 ENFP。
现在让我计算你和龙虾的配对度...

（生成报告）
```

### 最佳实践

#### 1. 定期测试

建议每月测试一次，观察配对度的变化：
- 刚安装 OpenClaw：测试一次，了解基准
- 使用 1 个月后：测试一次，观察变化
- 重要项目前：测试一次，优化协作模式

#### 2. 优化本地文件

为了让分析更准确，建议：
- **SOUL.md** - 更新你的性格、偏好、工作风格
- **USER.md** - 补充你的简历、经历、目标
- **MEMORY.md** - 记录重要决策、教训、洞察

#### 3. 结合报告行动

根据报告建议：
- **短期建议**（1-3 个月）：立即执行
- **中期建议**（3-6 个月）：规划执行
- **长期愿景**（6-12 个月）：战略规划
